## Pruebas unitarias

Consiste en probar la parte mas pequenia de un programa.
- Funcion
- Metodo
- Modulo
Se prueban de manera individual sin depender de otras partes del sistema
### Ejemplo
Funcion que calcule el promedio

1. Identificamos la unidad a probar: calcularPromedio()
2. Definir los casos de prueba: Entradas y salidas esperadas

| Caso | Entrada | Resultado esperado |
| ---- | ------- | ------------------ |
| 1    | 10,8,6  | 8                  |
| 2    | 5,5,5   | 5                  |
| 3    | 0,0,0   | 0                  |
3. Escribir el codigo de pruba

## Bug report
Estructura de reporte

| Campos                | Descripcion              |
| --------------------- | ------------------------ |
| ID                    | Identificador del error  |
| Titulo                | Nombre corto del bug     |
| Descripcion           | Explicacion del problema |
| Pasos para reproducir | Como ocurrem el problema |
| Resultado esperado    | Lo que deberia pasar     |
| Resultado obtenido    | Lo que realmente paso    |
| Severidad             | Impacto del error        |
| Prioridad             | Urgencia de correccion   |
| Evidencia             | Evidencia                |
